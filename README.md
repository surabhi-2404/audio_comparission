# Customer Churn Prediction Project
## Project Overview

This project analyzes a telecom customer dataset to understand customer behavior and predict whether a customer will churn (leave the service) or not.
It includes data preprocessing, visualization, and multiple machine learning models to compare performance and understand patterns affecting churn.
This notebook demonstrates fundamental data analysis + machine learning workflow, making it ideal for beginners learning predictive analytics.
